# TemperatureControl
 使用Arduino来控制温控器的温度。
Use Arduino to control the temperature of the thermostat.

## 指南 Guide

1. 按照引脚定义焊接按键到开发板引脚上。
2. 按照网上的教程，在Arduino IDE中添加 ESP32 开发板支持包。
3. 使用合宙 ESP32 C3 需要启用 USB CDC On Boot，将 Flash Mode 设置为 "DIO" 。
4. 下载程序，并用串口调试。
5. Enjoy.
